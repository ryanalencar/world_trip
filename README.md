## About the project
A landing page for a world trip website built with ChakraUI based on a Figma design 

## Technologies
- ReactJs
- NextJs
- Typescript
- ChakraUI

## Install

```sh
# Install the dependencies
yarn
```

## Usage

```sh
# Start the application
yarn dev
```
